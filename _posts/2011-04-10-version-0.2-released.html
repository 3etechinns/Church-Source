--- layout: post title: Version 0.2 Released categories: - release ---

The next release of Church Source is available now! This version's major driving feature was face recognition using Face.com's API. Currently, this can be used for face recognition during check in. However, in the future photos will also be used to help secure checkout procedures.

Also included were several bug fixes and Picnik integration to help crop faces from Household pictures.

To migrate from version 0.1 follow these steps:

  1. Download and extract Version 0.2.
  2. Move the directories static/uploads and static/cache from your old install to the V0.2 install.
  3. Run ./manage.py migrate

Download the tarball or go check out the source.

Please file issues if you run into any problems.
