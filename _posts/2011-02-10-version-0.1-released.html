After several weeks of coding and a half dozen uses I've come up with the first release of Church Source.

Download the tarball or go check out the source.

The first release focuses on check-in and integrates for use with touch screens. If you have a real touch screen like an iPad you can skip these features because the iPad already has features like a virtual keyboard. (However, I doubt you could use the iPad because you can't hook up a label printer to it.) The touch screen we use at Dominion Church is more simplistic and doesn't support swipes and such. That is what the touch screen features are geared towards.

I hope to write some more documentation soon on how to install Church Source and setup a check-in station, so check the Wiki frequently.

Also don't forget to file issues if you run into any problems.
